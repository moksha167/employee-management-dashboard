# Employee Management Dashboard


## Features

- Dashboard Overview

- Add Employee

- Edit Employee

- Delete Employee

- Search by Name

- Filter by Department

- Sort Employees (A-Z)

- Pagination

- LocalStorage Data Persistence

- Responsive Design



## Technologies Used

- HTML5

- CSS3

- JavaScript (ES6)

- LocalStorage



## How to Run

1. Download or clone the repository.

2. Open index.html in any web browser.
